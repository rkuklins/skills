# Skills

A private collection of Claude Code skills for handling repetitive tasks in my life.

Each skill lives in its own folder and teaches Claude how to handle a specific recurring workflow — so I don't have to explain the same process twice.

## Skills

| Skill | Description |
|-------|-------------|
| [weekly-brief](./weekly-brief/SKILL.md) | Generate a personal weekly status report from Linear projects |

## Usage

These skills are loaded automatically by Claude Code from `~/.cursor/skills`. To invoke a skill, use its name as a slash command — e.g. `/weekly-brief`.
